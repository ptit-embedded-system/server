# HOW TO RUN

- Create environment:

  Windows:

  ```bash
  python -m venv venv
  ```

  Linux:

  ```bash
  python3 -m venv venv
  ```

- Activate the environment:

  Linux/MacOS:

  ```bash
  . venv/bin/activate
  ```

  Windows:

  ```bash
  venv\Scripts\activate
  ```

- Install dependencies:

  ```bash
  python -m pip install --upgrade pip
  pip install -r requirements.txt
  ```

- Create a folder in project with name `uploads`

- Run flask:
  ```bash
  python app.py
  ```
